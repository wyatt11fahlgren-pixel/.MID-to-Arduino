# .MID-to-Arduino
this turns a .mid file into IDE code for an arduino to play with buzzers.

just open with your browser, add the .mid file and press convert.
copy and paste the code given

----------------------------------------------------------------------------------------------------

extra tip! if you click into the code box, press ctrl+A to highlight all, and then Ctrl+C to copy it.
this can make realy long songs easer to copy and paste
